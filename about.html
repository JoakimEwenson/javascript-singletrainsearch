Allmän information

Detta är ett hobbyprojekt signerat Joakim Ewenson och bör inte ses som något annat än just det, ett hobbyprojekt.

Den data som detta hobbyprojekt genererar kommer från Trafikverkets API för öppna data och det är även de som får stå för garantierna kring dess giltighet. Alla former av nya tider som presenteras är att ses som preliminära och kan komma att ändras med kort varsel, det lämnas inga garantier för eventuella ändringar som resulterar i missade tåg och annat elände.

Jag har gjort mitt bästa för att göra en någorlunda bra presentation utav den information som finns att tillgå baserat på min erfarenhet från järnvägen som sektor. Vissa saker kan vara fullt naturliga för mig och helt orimliga för andra, just därför uppskattar jag all form av feedback som kan komma in!

Trafikverkets öppna data

Informationen som presenteras i den här tjänsten kommer som sagt från Trafikverkets öppna API med de brister och tillkortakommanden som den tjänsten har. Jag har, efter bästa förmåga, försökt att presentera det hela på ett enligt mig bra sätt.

Mer information om Trafikverkets öppna API och den data som finns tillgängligt i det finns på https://api.trafikinfo.trafikverket.se/.

Tekniken bakom tjänsten

Denna tjänst är baserad på HTML, CSS (Bootstrap) och JavaScript utan några ramverk inblandade. Sidan ritas upp i HTML och sedan påverkas DOMen via asynkron JavaScript som uppdaterar innehållet på ett förbestämt intervall. I framtiden hoppas jag kunna bygga om projektet så att det istället förlitar sig på från Trafikverket pushad data istället, för att spara på allas bandbredd och datorer.

Observera att tjänsten sparar några få spår av data i din webbläsarsession. Det som sparas är sökt tågnummer och/eller stationssignatur samt sökt datum. Detta för att kunna återhämta data vid omladdning utav sidan. Mer information än så är inte av intresse och inget sparas efter att du stängt webbläsarfönstret/webbläsarfliken.
